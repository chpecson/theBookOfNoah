# three.js use cases

# API
## terminology
  - plane: 2d rectangle that serves as ground area
  - cube: 3d cube
  - sphere: 3d sphere
  - camera: dtermines what you'll see in the output
  - axes: x,y,z; helpful debugging tool to see where the objects are rendered in 3d space
